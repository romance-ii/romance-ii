How to get WebGL for Microsoft Internet Explorer

What is WebGL?

WebGL is the graphics system that lets your web browser show 3D graphics.

What is Microsoft Internet Explorer?

Internet Explorer is a web browser. Its icon is a big blue e.

Why can't I play this game in Internet Explorer?

Microsoft has not provided Internet Explorer with the capability to display 3D graphics without using a plug-in.

Can I get a better browser?

Internet Explorer is not very good at working with web pages. If you possibly can, the best thing you can do is download another browser. Here is a bit of an explanation; we recommend Firefox.

What if I can't upgrade to a better browser?

You will need a plug-in to view 3D graphics.
