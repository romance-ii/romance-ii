Technology Demo

What is this?

This page is a Technology Demo. This is not a fully-featured or playable game. In fact, this is being used as a scratch pad by the programmers. So, it's quite possible that it might not work; if we are testing things like the way that bugs are handled, in fact, we may have broken it, intentionally.

The Framerate Test

The current test is to judge what device(s) we might be able to support. Here's how it works:

  • Try to visit the page linked below.
  • If you get a message that your computer can't start the demo, please let us know this information:
    • What kind of device? (e.g. computer; smartphone; tablet; videogame)
    • What kind of system is it? (e.g. Samsung Galaxy S-III Android, MicroSoft Windows 8, Fedora 19
    • If it's a PC, what kind of graphics processor do you have (if you know)? (e.g. ATI Radeon, nVidia 860)
    • What web browser are you using? (e.g. Firefox 20, Safari Mobile)
  • You might want to visit get.webgl.com and see if there's an easy way to enable WebGL in your browser
  • If your device/browser isn't capable right now, don't worry too much. It's possible that it will get an update before the game is ready to be played; or, we may be able to do something on our end to make it work. It is still helpful for us to know that you might have trouble, so that we can try to make it work.
  • If you can start the test, click Log In, enter any name you like, and wait a while. Please don't change windows or tabs until you finish the test, because this will make it slow down and give you a poorer score. Just stare at the screen for about a minute. There really won't me much (if anything) to see here, except the smiley face.
  • You'll eventually see a smiley appear in the top-right corner; after a while, a button will appear below the big display panel. (You might need to scroll down…) When you click it, a bunch of useful technical information will be printed. You can either click that message to send it by eMail automatically, or cut-n-paste it into an eMail message in your own way.
  • The Player Name you enter is completely ignored right now, but it will be included in the report text generated. So, if you would like to tag different devices, feel free to enter something there. (Example: chrome on George's iPad)
  • If you don't Log Out before leaving the page, you'll get a warning … you can safely ignore it.

Please send us an eMail with your report! violetvolts-frametest@star-hope.org is the address.
